Getting Started

Quick start guide to integrate Ape Design System into your project.

1. Include the CSS File

Add the Ape Design System CSS file to your HTML:

<link rel="stylesheet" href="assets/ape-design.css">

Or use it from a CDN (if hosted):

<link rel="stylesheet" href="https://your-domain.com/assets/ape-design.css">

2. Set a Theme

Add a data-theme attribute to your <body> tag:

<body data-theme="cyan">
  <!-- Your content -->
</body>

Available themes: cyan, purple, green, orange, pink

3. Use Semantic HTML

The design system works best with semantic HTML5 elements:

<section>
  <div class="container">
    <h2>Section Title</h2>
    <article>
      <h3>Card Title</h3>
      <p>Card content</p>
      <footer>
        <button>Action</button>
      </footer>
    </article>
  </div>
</section>

Theme Switching

Switch themes dynamically using JavaScript or CSS.

JavaScript Method

Change themes programmatically:

// Set theme
document.body.setAttribute('data-theme', 'purple');

// Get current theme
const currentTheme = document.body.getAttribute('data-theme');

Dropdown Theme Switcher

Use the included dropdown theme switcher (see index.html for implementation):

<div class="theme-switcher">
  <div class="theme-dropdown">
    <button class="theme-dropdown-button" id="themeButton">
      <span id="themeButtonText">Cyan Nexus</span>
    </button>
    <div class="theme-dropdown-menu" id="themeMenu">
      <div class="theme-option active" data-theme="cyan">
        <div class="theme-option-color"></div>
        <div class="theme-option-label">
          <span class="theme-option-name">Cyan Nexus</span>
        </div>
      </div>
      <!-- More theme options -->
    </div>
  </div>
</div>

CSS Variables

All aspects of the design system can be customized using CSS variables.

Color Variables

:root {
  /* Background Colors */
  --bg-primary: #000000;
  --bg-secondary: #0a0a0f;
  --bg-tertiary: #151520;
  
  /* Text Colors */
  --text-primary: #ffffff;
  --text-secondary: #e5e7eb;
  --text-muted: #9ca3af;
  --text-dim: #6b7280;
  
  /* Accent Colors */
  --accent-primary: #00ffff;
  --accent-secondary: #00ff88;
  --accent-tertiary: #ff00ff;
  --accent-glow: rgba(0, 255, 255, 0.5);
}

Glass Effect Variables

:root {
  --glass-bg: rgba(255, 255, 255, 0.03);
  --glass-bg-hover: rgba(255, 255, 255, 0.05);
  --glass-border: rgba(255, 255, 255, 0.1);
  --glass-border-accent: rgba(0, 255, 255, 0.3);
  --glass-blur: blur(20px);
  --glass-blur-strong: blur(40px);
}

Components

Built-in components with glass morphism effects.

Glass Cards (Articles)

Use <article> for glass cards:

<article>
  <h3>Card Title</h3>
  <p>Card content goes here.</p>
  <footer>
    <button>Action</button>
  </footer>
</article>

Cards automatically have glass effects, hover animations, and footer positioning.

Buttons

Three button styles available:

<button>Primary Button</button>
<button class="outline">Outline Button</button>
<button class="full">Full Width Button</button>

<!-- Or use anchor tags -->
<a href="#" role="button">Link Button</a>

Grid Layouts

Responsive grid system:

<div class="grid">
  <article>Card 1</article>
  <article>Card 2</article>
  <article>Card 3</article>
</div>

<!-- Compact grid for smaller items -->
<div class="grid compact">
  <article>Item 1</article>
  <article>Item 2</article>
</div>

Sections

Use <section> for content blocks:

<section>
  <div class="container">
    <h2>Section Title</h2>
    <p>Section content</p>
  </div>
</section>

<!-- Secondary section with dark background -->
<section class="secondary">
  <div class="container">
    <h2>Secondary Section</h2>
  </div>
</section>

Customization

Create your own theme or customize existing ones.

Creating a Custom Theme

Add a new theme by creating a new data-theme selector:

[data-theme="custom"] {
  --accent-primary: #ff0000;
  --accent-secondary: #ff3333;
  --accent-tertiary: #cc0000;
  --accent-glow: rgba(255, 0, 0, 0.6);
  --glass-border-accent: rgba(255, 0, 0, 0.4);
}

Then use it: <body data-theme="custom">

Overriding Variables

Override any variable in your own CSS:

:root {
  --accent-primary: #your-color;
  --glass-bg: rgba(255, 255, 255, 0.05);
  --border-radius: 16px;
  /* Override any variable */

Disabling CRT Effects

To remove the scanline effect, override the body::before pseudo-element:

body::before {
  display: none;
}

          </article>
        </div>
      </div>
    </section>
  </main>
</body>
</html>
