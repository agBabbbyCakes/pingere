Color Variables

Control all colors in the design system. Background colors, text colors, and accent colors can all be customized.

Background Colors

Primary background colors for the page and components.

--bg-primary

Default: #000000

Usage: Main page background color. This is the darkest background layer.

background: var(--bg-primary); 
/* Override example */
:root {
  --bg-primary: #0a0a0a; /* Slightly lighter black */
}

--bg-secondary

Default: #0a0a0f

Usage: Secondary background for sections and panels. Slightly lighter than primary.

background: var(--bg-secondary);

--bg-tertiary

Default: #151520

Usage: Tertiary background for nested components or elevated surfaces.

background: var(--bg-tertiary);

Text Colors

Hierarchy of text colors for different content levels.

--text-primary

Default: #ffffff

Usage: Primary text color for headings and important content. Highest contrast.

color: var(--text-primary); - This is primary text

--text-secondary

Default: #e5e7eb

Usage: Secondary text for body content and paragraphs. Good readability.

color: var(--text-secondary); - This is secondary text

--text-muted

Default: #9ca3af

Usage: Muted text for less important information, captions, and metadata.

color: var(--text-muted); - This is muted text

--text-dim

Default: #6b7280

Usage: Dim text for disabled states or very low-priority content.

color: var(--text-dim); - This is dim text

Accent Colors

Theme-specific accent colors that change based on the selected theme.

--accent-primary

Default (Cyan): #00ffff

Usage: Primary accent color for links, buttons, highlights, and interactive elements. This is the main theme color.

color: var(--accent-primary); - Accent text

/* Theme-specific overrides */
[data-theme="purple"] {
  --accent-primary: #a855f7;
}

[data-theme="green"] {
  --accent-primary: #00ff88;
}

--accent-secondary

Default: #00ff88

Usage: Secondary accent for complementary elements or hover states.

color: var(--accent-secondary);

--accent-tertiary

Default: #ff00ff

Usage: Tertiary accent for special highlights or decorative elements.

color: var(--accent-tertiary);

--accent-glow

Default (Cyan): rgba(0, 255, 255, 0.5)

Usage: Glow color for shadows, text shadows, and hover effects. Used in box-shadow and text-shadow.

box-shadow: 0 0 20px var(--accent-glow);

/* Usage in CSS */
.element {
  box-shadow: 0 0 20px var(--accent-glow);
  text-shadow: 0 0 10px var(--accent-glow);
}

Glass Effect Variables

Control the glass morphism effects throughout the design system. These variables create the frosted glass aesthetic.

Glass Backgrounds

--glass-bg

Default: rgba(255, 255, 255, 0.03)

Usage: Base glass background for cards, panels, and components. Very subtle white overlay.

background: var(--glass-bg); 
/* Customize glass opacity */
:root {
  --glass-bg: rgba(255, 255, 255, 0.05); /* More opaque */
}

--glass-bg-hover

Default: rgba(255, 255, 255, 0.05)

Usage: Glass background for hover states. Slightly more opaque than base.

background: var(--glass-bg-hover); (hover state)

Glass Borders

--glass-border

Default: rgba(255, 255, 255, 0.1)

Usage: Standard glass border color. Subtle white border for glass elements.

border: 1px solid var(--glass-border);

--glass-border-accent

Default (Cyan): rgba(0, 255, 255, 0.3)

Usage: Accent border color for hover states and active elements. Theme-specific.

border: 1px solid var(--glass-border-accent); (hover/active)

Backdrop Blur

--glass-blur

Default: blur(20px)

Usage: Standard backdrop blur for glass effects. Used with backdrop-filter.

backdrop-filter: var(--glass-blur); 
/* Usage */
.element {
  backdrop-filter: var(--glass-blur);
  -webkit-backdrop-filter: var(--glass-blur); /* Safari */
}

--glass-blur-strong

Default: blur(40px)

Usage: Stronger blur for headers, modals, or elements that need more separation.

backdrop-filter: var(--glass-blur-strong);

Border Variables

Control border styles, widths, and radius throughout the design system.

Border Properties

--border-width

Default: 1px

Usage: Standard border width for all borders.

border-width: var(--border-width);

--border-style

Default: solid

Usage: Border style. Can be changed to dashed, dotted, etc.

:root {
  --border-style: dashed; /* Change to dashed borders */
}

--border-color

Default: var(--glass-border)

Usage: Default border color. References glass border variable.

--border-accent

Default: var(--glass-border-accent)

Usage: Accent border color. References glass border accent variable.

Border Radius

--border-radius

Default: 12px

Usage: Standard border radius for cards, buttons, and rounded elements.

border-radius: var(--border-radius); 
/* Customize roundness */
:root {
  --border-radius: 16px; /* More rounded */
  --border-radius: 8px;  /* Less rounded */
}

--border-radius-sm

Default: 8px

Usage: Small border radius for compact elements or inline code.

border-radius: var(--border-radius-sm);

--border-radius-lg

Default: 16px

Usage: Large border radius for prominent elements or hero sections.

border-radius: var(--border-radius-lg);

Shadow & Glow Variables

Control shadows and glow effects for depth and lighting.

Shadows

--shadow-sm

Default: 0 2px 8px rgba(0, 0, 0, 0.3)

Usage: Small shadow for subtle elevation. Use on buttons and small cards.

box-shadow: var(--shadow-sm);

--shadow-md

Default: 0 4px 16px rgba(0, 0, 0, 0.4)

Usage: Medium shadow for cards and panels. Standard elevation.

box-shadow: var(--shadow-md);

--shadow-lg

Default: 0 8px 32px rgba(0, 0, 0, 0.5)

Usage: Large shadow for modals, dropdowns, and elevated surfaces.

box-shadow: var(--shadow-lg);

Glows

--glow-accent

Default: 0 0 20px var(--accent-glow)

Usage: Standard accent glow for hover states and active elements.

box-shadow: var(--glow-accent); 
/* Usage */
button:hover {
  box-shadow: var(--glow-accent);
}

--glow-accent-strong

Default: 0 0 40px var(--accent-glow)

Usage: Strong accent glow for prominent elements or focus states.

box-shadow: var(--glow-accent-strong);

--glow-inner

Default: inset 0 0 20px rgba(0, 255, 255, 0.1)

Usage: Inner glow for depth. Creates a subtle inner light effect.

box-shadow: var(--glow-inner);

Typography Variables

Control fonts, sizes, weights, and text spacing.

Font Families

--font-family

Default: 'SF Mono', 'Monaco', 'Inconsolata', 'Fira Code', 'Courier New', monospace

Usage: Monospace font for code, technical content, and brand elements.

font-family: var(--font-family);

--font-family-body

Default: system-ui, -apple-system, sans-serif

Usage: Body font for paragraphs and general content. Uses system fonts.

font-family: var(--font-family-body);

Font Sizes

Complete size scale from extra small to extra extra large.

--font-size-xs

Default: 0.75rem (12px)

Usage: Extra small text for captions, labels, and fine print.

font-size: var(--font-size-xs);

--font-size-sm

Default: 0.875rem (14px)

Usage: Small text for secondary content and metadata.

font-size: var(--font-size-sm);

--font-size-base

Default: 1rem (16px)

Usage: Base font size for body text and paragraphs.

font-size: var(--font-size-base);

--font-size-lg

Default: 1.125rem (18px)

Usage: Large text for emphasized content.

font-size: var(--font-size-lg);

--font-size-xl

Default: 1.25rem (20px)

Usage: Extra large text for subheadings.

font-size: var(--font-size-xl);

--font-size-2xl

Default: 1.5rem (24px)

Usage: 2X large for h3 headings.

font-size: var(--font-size-2xl);

--font-size-3xl

Default: 2rem (32px)

Usage: 3X large for h2 headings.

font-size: var(--font-size-3xl);

--font-size-4xl

Default: 2.5rem (40px)

Usage: 4X large for large h2 headings.

font-size: var(--font-size-4xl);

--font-size-5xl

Default: 3.5rem (56px)

Usage: 5X large for h1 hero headings.

font-size: var(--font-size-5xl);

Font Weights

--font-weight-normal

Default: 400

Usage: Normal font weight for body text.

--font-weight-medium

Default: 500

Usage: Medium font weight for buttons and emphasized text.

--font-weight-bold

Default: 700

Usage: Bold font weight for headings and strong emphasis.

Line Height & Letter Spacing

--line-height-tight

Default: 1.2

Usage: Tight line height for headings.

--line-height-normal

Default: 1.6

Usage: Normal line height for body text and paragraphs.

--letter-spacing-tight

Default: -0.02em

Usage: Tight letter spacing for large headings (h1).

Spacing Variables

Consistent spacing scale for padding, margin, and gaps.

Spacing Scale

All spacing values use a consistent scale from extra small to 4X large.

--spacing-xs

Default: 0.25rem (4px)

Usage: Extra small spacing for tight layouts.

padding: var(--spacing-xs);

--spacing-sm

Default: 0.5rem (8px)

Usage: Small spacing for compact elements.

padding: var(--spacing-sm);

--spacing-md

Default: 1rem (16px)

Usage: Medium spacing - the base unit. Most common spacing value.

padding: var(--spacing-md);

--spacing-lg

Default: 1.5rem (24px)

Usage: Large spacing for card padding and section gaps.

padding: var(--spacing-lg);

--spacing-xl

Default: 2rem (32px)

Usage: Extra large spacing for container padding and major gaps.

padding: var(--spacing-xl);

--spacing-2xl

Default: 3rem (48px)

Usage: 2X large spacing for major section spacing.

padding: var(--spacing-2xl);

--spacing-3xl

Default: 4rem (64px)

Usage: 3X large spacing for hero sections.

--spacing-4xl

Default: 6rem (96px)

Usage: 4X large spacing for major page sections.

Usage Examples

/* Padding */
.element {
  padding: var(--spacing-md);
  padding-top: var(--spacing-lg);
}

/* Margin */
.element {
  margin: var(--spacing-xl) 0;
  margin-bottom: var(--spacing-2xl);
}

/* Gap in flexbox/grid */
.container {
  display: flex;
  gap: var(--spacing-lg);
}

Layout Variables

Control container widths and layout spacing.

Container Settings

--container-width

Default: 1200px

Usage: Maximum width for main content containers. Centers content on large screens.

max-width: var(--container-width); 
/* Customize container width */
:root {
  --container-width: 1400px; /* Wider */
  --container-width: 960px;  /* Narrower */
}

--container-padding

Default: var(--spacing-xl) (2rem)

Usage: Horizontal padding for containers. Provides side spacing on all screen sizes.

.container {
  max-width: var(--container-width);
  padding: 0 var(--container-padding);
  margin: 0 auto;
}

--section-padding

Default: var(--spacing-4xl) 0 (6rem 0)

Usage: Vertical padding for major sections. Top and bottom spacing.

section {
  padding: var(--section-padding);
}

Transition Variables

Control animation timing and easing for smooth interactions.

Transition Timing

--transition-fast

Default: 0.15s ease

Usage: Fast transitions for hover states and quick interactions.

button {
  transition: all var(--transition-fast);
}

--transition-base

Default: 0.2s ease

Usage: Standard transition timing. Most common for hover and state changes.

--transition-slow

Default: 0.3s ease

Usage: Slow transitions for major state changes or page transitions.

Usage Examples

/* Standard hover transition */
.element {
  transition: all var(--transition-base);
}

.element:hover {
  transform: translateY(-2px);
  box-shadow: var(--glow-accent);
}

/* Fast color change */
.link {
  transition: color var(--transition-fast);
}

/* Slow transform */
.modal {
  transition: transform var(--transition-slow);
}

CRT Effect Variables

Control the retro CRT scanline effects.

Scanline Effects

--crt-scanline

Default: repeating-linear-gradient(0deg, rgba(0, 0, 0, 0.15), rgba(0, 0, 0, 0.15) 1px, transparent 1px, transparent 2px)

Usage: Creates horizontal scanlines for CRT effect. Applied to body::before pseudo-element.

CRT scanline effect overlay

/* Applied automatically to body */
body::before {
  content: '';
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: var(--crt-scanline);
  pointer-events: none;
  z-index: 9999;
  opacity: 0.3;
  mix-blend-mode: overlay;
}

/* Disable CRT effect */
body::before {
  display: none;
}

--crt-flicker

Default: 0.15s

Usage: Timing for CRT flicker animations (if implemented).

Customizing CRT Effects

/* More visible scanlines */
:root {
  --crt-scanline: repeating-linear-gradient(
    0deg,
    rgba(0, 0, 0, 0.3),  /* Darker lines */
    rgba(0, 0, 0, 0.3) 1px,
    transparent 1px,
    transparent 2px
  );
}

/* Thicker scanlines */
:root {
  --crt-scanline: repeating-linear-gradient(
    0deg,
    rgba(0, 0, 0, 0.15),
    rgba(0, 0, 0, 0.15) 2px,  /* 2px lines */
    transparent 2px,
    transparent 4px
  );
}

Quick Reference

Common variable overrides and customization patterns.

Common Customizations

/* Change accent color globally */
:root {
  --accent-primary: #ff6b35;
  --accent-glow: rgba(255, 107, 53, 0.6);
  --glass-border-accent: rgba(255, 107, 53, 0.4);
}

/* Make glass more opaque */
:root {
  --glass-bg: rgba(255, 255, 255, 0.08);
  --glass-bg-hover: rgba(255, 255, 255, 0.12);
}

/* Increase border radius */
:root {
  --border-radius: 16px;
  --border-radius-lg: 24px;
}

/* Tighter spacing */
:root {
  --spacing-md: 0.75rem;
  --spacing-lg: 1.25rem;
}

/* Wider container */
:root {
  --container-width: 1400px;
}

Creating a Custom Theme

[data-theme="custom"] {
  /* Accent colors */
  --accent-primary: #your-color;
  --accent-secondary: #your-secondary;
  --accent-tertiary: #your-tertiary;
  --accent-glow: rgba(/* your color with opacity */);
  
  /* Glass borders */
  --glass-border-accent: rgba(/* your color with opacity */);
}

/* Usage */
<body data-theme="custom">
