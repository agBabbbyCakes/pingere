Ape Framework is an easy-to-use Web3 development tool. Users can compile, test, and interact with smart contracts all in one command line session. With our modular plugin system, Ape supports multiple contract languages and chains.

Read the Docs Back to Home

Features

Everything you need for smart contract development in Python.

Multi-Chain Support

Support for Ethereum, Polygon, Arbitrum, and more through our plugin system. Switch chains seamlessly.

Contract Compilation

Compile Solidity, Vyper, and other contract languages with built-in compilers and plugins.

Testing Framework

Write tests in Python using pytest. Test your contracts with comprehensive assertion libraries.

Interactive Console

Use Ape Console for an interactive Python REPL with your contracts and chain data pre-loaded.

Plugin System

Extend Ape with plugins for new chains, compilers, and development tools.

Pythonic API

Work with contracts and blockchain data using clean, Pythonic APIs. No need to learn new languages.

Get Started

Install Ape Framework and start building in minutes.
