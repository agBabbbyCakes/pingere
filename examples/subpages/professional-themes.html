How to Use

Switching themes is simple - just add a data-theme attribute to your <body> tag.

Setting a Theme

Add the theme attribute to your HTML:

<body data-theme="cyan">
  <!-- Your content -->
</body>

Switching Themes with JavaScript

Change themes dynamically:

// Set theme
document.body.setAttribute('data-theme', 'green');

// Get current theme
const currentTheme = document.body.getAttribute('data-theme');

Theme Switcher Component

Use the included dropdown theme switcher for user-controlled theme switching. See the documentation for implementation details.

Customization

All themes use CSS variables, making it easy to create your own theme variations.

Creating Custom Themes

Add a new theme by creating a new data-theme selector:

[data-theme="custom"] {
  --accent-primary: #ff0000;
  --accent-secondary: #ff3333;
  --accent-tertiary: #cc0000;
  --accent-glow: rgba(255, 0, 0, 0.6);
  --glass-border-accent: rgba(255, 0, 0, 0.4);
}

Theme Variables

Each theme defines these key variables:
