Glass Morphism

Subtle frosted glass effects with backdrop blur. Cards and panels have a premium, professional aesthetic perfect for enterprise applications.

Back to Home View Docs

Overview

Glass morphism is a modern design trend that creates a frosted glass effect using semi-transparent backgrounds, backdrop filters, and subtle borders. The Ape Design System implements this effect with professional, enterprise-ready styling.

Key Characteristics

  • Semi-transparent backgrounds with backdrop blur
  • Subtle borders for depth and definition
  • Smooth hover transitions
  • Professional opacity levels
  • Fully customizable via CSS variables

Examples

Glass morphism is applied automatically to cards, buttons, and other components.

Glass Cards

Cards automatically have glass effects with backdrop blur and subtle borders. Hover to see the enhanced effect.

Glass Buttons

Buttons feature glass backgrounds with accent borders. Perfect for primary actions.

Implementation

Glass morphism effects are built into the design system components.

Using Glass Cards

Simply use semantic HTML - glass effects are automatic:

<article>
  <h3>Card Title</h3>
  <p>Card content goes here.</p>
  <footer>
    <button>Action</button>
  </footer>
</article>

Glass Buttons

Buttons automatically have glass styling:

<button>Primary Button</button>
<button class="outline">Outline Button</button>

Customization

Control glass morphism effects with CSS variables.

Glass Effect Variables

Customize the glass appearance:

:root {
  --glass-bg: rgba(255, 255, 255, 0.03);
  --glass-bg-hover: rgba(255, 255, 255, 0.05);
  --glass-border: rgba(255, 255, 255, 0.1);
  --glass-border-accent: rgba(0, 255, 255, 0.3);
  --glass-blur: blur(20px);
  --glass-blur-strong: blur(40px);
}

Adjusting Opacity

Make glass effects more or less transparent:

:root {
  --glass-bg: rgba(255, 255, 255, 0.05); /* More opaque */
  --glass-bg-hover: rgba(255, 255, 255, 0.08);
}
