Grid Overlays

Optional subtle grid overlays with masked radial gradients for atmospheric depth. Professional and understated visual effects that enhance your design without overwhelming it.

Back to Home View Docs

Overview

Grid overlays provide a subtle, professional backdrop that adds depth and visual interest to your pages. The grid pattern uses masked radial gradients to create a sophisticated, understated effect that works perfectly for enterprise applications.

Key Features

  • Subtle grid pattern with customizable spacing
  • Masked radial gradients for atmospheric depth
  • Low opacity for professional appearance
  • Easy to enable or disable per section
  • Fully customizable via CSS variables

Usage Examples

Grid overlays can be added to any section using the .grid-overlay class.

Basic Grid Overlay

Add a grid overlay to any section:

<section class="hero">
  <div class="grid-overlay"></div>
  <h1>Hero Title</h1>
</section>

Secondary Section with Grid

Grid overlays work especially well with secondary sections:

<section class="secondary">
  <div class="grid-overlay"></div>
  <div class="container">
    <h2>Section Title</h2>
  </div>
</section>

Customization

Customize grid overlays using CSS variables in your theme or custom stylesheet.

CSS Variables

Control grid appearance with these variables:

:root {
  --accent-glow: rgba(0, 255, 255, 0.5);
  /* Grid overlay uses --accent-glow for line color */
}

Grid Spacing

The default grid uses 40px spacing. You can override this in your CSS:

.grid-overlay {
  background:
    linear-gradient(to right, var(--accent-glow) 1px, transparent 1px) 0 0 / 60px 60px,
    linear-gradient(to bottom, var(--accent-glow) 1px, transparent 1px) 0 0 / 60px 60px;
}
