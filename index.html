APE DESIGN SYSTEM

Futuristic dark glass themes with neon accents. Space-age aesthetics meets CRT nostalgia. Fully customizable with extensive CSS variables.

Get Started View Docs

Core Features

Everything you need to build futuristic, glass-morphism interfaces with neon accents.

5 Dark Themes

Five distinct dark themes with glass effects and vibrant neon accents. Switch themes instantly with data attributes.

Glass Morphism

Beautiful frosted glass effects with backdrop blur. Cards and panels have that space-age glass aesthetic.

CRT Effects

Subtle scanline effects and glows that evoke retro CRT monitors. Perfect for that futuristic retro vibe.

5 Distinct Themes

Switch between themes using the theme switcher in the header. Each theme has unique neon accents.

Cyan Nexus

Default theme with vibrant cyan accents. Perfect for tech and developer tools.

Use: data-theme="cyan"

Purple Matrix

Deep purple with magenta highlights. Matrix-inspired with a modern twist.

Use: data-theme="purple"

Green Terminal

Classic terminal green with lime accents. Nostalgic yet futuristic.

Use: data-theme="green"

Orange Fusion

Warm orange with amber glow. Energy and warmth in a dark interface.

Use: data-theme="orange"

Pink Neon

Vibrant pink with magenta pulse. Bold and energetic.

Use: data-theme="pink"

Customizable

All themes use CSS variables. Edit root variables to create your own theme variations.

See ape-design.css for variables
