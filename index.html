Professional dark design system with glass morphism effects. Enterprise-ready themes built for modern web applications. Fully customizable with extensive CSS variables.

Core Features

Everything you need to build professional, enterprise-ready interfaces with glass morphism effects.

2 Professional Themes

Two carefully crafted themes designed for enterprise use. Professional Blue for B2B products, Premium Emerald for fintech and growth platforms. Switch themes instantly with data attributes.

Glass Morphism

Subtle frosted glass effects with backdrop blur. Cards and panels have a premium, professional aesthetic perfect for enterprise applications.

Grid Overlays

Optional subtle grid overlays with masked radial gradients for atmospheric depth. Professional and understated.

Two Professional Themes

Choose the theme that best fits your product. Both are designed for professional, enterprise-ready applications.

Professional Blue

Modern SaaS aesthetic - clean, trustworthy, enterprise-ready. Perfect for B2B products, developer tools, and enterprise software. Conveys reliability and professionalism.

Use: data-theme="cyan"

Premium Emerald

Sophisticated green - growth, innovation, premium feel. Perfect for fintech, crypto, growth products, and premium services. Conveys success and forward-thinking.

Use: data-theme="green"

Fully Customizable

All themes use CSS variables. Edit root variables to create your own theme variations. Easy to extend for brand-specific colors.

See ape-design.css for variables
